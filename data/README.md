# Data

The original study used public NHTSA National Automotive Sampling System / Crashworthiness Data System (NASS/CDS) crash records for 2009-2015. The raw NASS/CDS records are not copied here because they are public source data maintained by NHTSA.

This package includes a modern CSV extract of one Dropbox workbook found during audit: `ML Accuracy and Var.Imp/Different Algorithms.xlsx`. The workbook records model-performance summary results for RF, kNN, SVM, and POLR runs under alternative feature sets. No raw crash-level analysis table or executable modeling script was found locally.

Public source pointers are listed in `metadata/PUBLIC_REFERENCE_LINKS.csv`.
